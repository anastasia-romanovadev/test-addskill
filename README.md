# AdSkill Frontend Test Task

Тестовое задание: реализация дашборда с офферами, таблицей, фильтрами и карточками по макету Figma на стеке React + TypeScript + Redux Toolkit + Vite.

## Стек

- React 18
- TypeScript
- Vite
- Redux Toolkit (@reduxjs/toolkit, react-redux)
- SCSS-модули для стилизации

## Функциональность

- Загрузка списка офферов из мок-данных.
- Таблица офферов с сортировкой по колонкам и выбором строк.
- Переключение между десктопной таблицей и мобильными карточками.
- Фильтры по дате, офферам, валюте и площадкам (UI-часть).
- Виджет метрик (расход, клики, CPC, CPA) по выбранному офферу.
- Карточка баланса и детальная информация по офферу.
- Таблица рекламных площадок для выбранного оффера.

## Запуск проекта

npm install
npm run dev

## Структура

- `src/components/dashboard` — основные компоненты дашборда (таблица, карточки, фильтры, метрики, виджеты).
- `src/features` — Redux-slices и логика работы с офферами.
- `src/utils/formatters` — форматирование дат, валют и числовых значений.
- `src/mocks` — мок-данные для офферов.

## Примечания

- Состояния `loading` и `error` обрабатываются в компоненте `Dashboard`.
- Стили вынесены в SCSS-файлы рядом с компонентами и максимально приближены к предоставленному макету Figma.
